SignalBox Dashboard Redesign

Three proposals for a data-dense, long-scroll dashboard. Pick the one that feels right.

Proposal A: Long-Scroll + Slide-Over

Single scrollable page with 6 zones. Click any project to open a slide-over panel from the right. No URL routing -- pure state. Settings via Cmd+K.

6-zone scroll slide-over panel comparison grid vertical feed priority filter countdown timer

Proposal B: Long-Scroll + Hash Routing

Same long-scroll home, but clicking a project navigates to #/project/name (dedicated page). Browser back/forward works. URLs are shareable and bookmarkable. Includes influencer leaderboard.

hash routing shareable urls browser history influencer board full project page crossfade transitions

Hybrid: Slide-Over + Hash Routing + Full Pages

Best of both worlds. Click a project on home for a quick slide-over preview without losing scroll position. Hit "Full Report" in the slide-over to navigate to a dedicated project page with shareable URL. Browser history works. Includes influencer leaderboard.

slide-over preview hash routing shareable urls influencer board full project page quick peek + deep dive
