# SignalBox Design Brief

---

## 1. IDENTITY

**Product name:** SignalBox
**One-line purpose:** On-chain sentiment oracle for crypto projects
**Who uses it:** Hackathon judges, then crypto project teams
**When they use it:** At desk, reviewing data, assessing project health
**Mood in 3 words:** precise, restrained, institutional

---

## 2. COLOR

**Brand color:** #E0743A (muted orange -- Chainlink-adjacent)

**Background:** Dark (#0C0C0E)

**Gray scale:** Neutral grays (no undertone)

**Data colors (5 categories):**

| Category | Color | Hex |
|----------|-------|-----|
| Praise | muted teal | #5A9E8F |
| Bug | muted rose | #9E6B6B |
| Complaint | muted rose | #9E6B6B |
| Feature request | muted slate | #7B7E9E |
| Question | muted sand | #9E8F6B |

> All at ~40% saturation. They distinguish categories, not scream.

---

## 3. TYPOGRAPHY

**Heading font:** Space Grotesk
**Body font:** DM Sans

**Size scale:** Compact (body 13px, small 11px)

**Heading sizes:**
- Page title: 24px
- Section title: 16px
- Card title: 14px

---

## 4. LAYOUT

**Primary layout:** Single column (scrolls vertically)

**Max content width:** 1200px

**Card style:** Bordered (1px border, no shadow)

**Border radius:** Soft (4px)

---

## 5. SPACING

**Density:** Standard (24px gaps, 16px padding)

---

## 6. NAVIGATION

**Primary navigation:** None (single page scroll + command palette)

**Top-level pages:** 1 (home scroll) + project detail via slide-over

---

## 7. COMPONENTS

**Data display:** Cards, Stat numbers, Tags/badges, Timeline/feed, Charts (sparklines only)
**Inputs:** Dropdowns/selects, Buttons (primary + ghost), Search
**Overlays:** Slide-over panel, Command palette, Toast
**State indicators:** Loading (skeleton), Empty state, Error state

---

## 8. ANIMATION

**Motion philosophy:** Minimal (150ms fades on hover/state change)

---

## 9. WHAT THIS UI IS NOT

1. NOT like: a crypto trading terminal with neon and gradients
2. NOT like: a SaaS marketing page with large hero and CTAs
3. NOT like: a playful consumer app with rounded cards and emoji

---

## 10. REFERENCE

1. Linear.app (clean, dense, monochrome with single accent)
2. Vercel dashboard (flat cards, restrained color, data-forward)
3. Bloomberg Terminal (data density, institutional feel)
